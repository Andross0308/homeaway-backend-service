# homeaway-backend-service
A custom Java backend application for student service management. Built from scratch without java.util to demonstrate mastery of Data Structures, Memory Management, and Geometric Search.

Homeaway From Home - Academic Services Management
This system manages a geographical database of services (Eating, Lodging and Leisure) for international students.


**Technical Challenges & Implementation**:

_**No Standard Libraries**_: Full implementation of data structures (Lists, Maps, etc.) without using java.util, ensuring complete control over algorithmic complexity.

_**Geolocation**_: System based on Bounding Boxes (latitude/longitude represented by high-precision integers) for service validation.

**Notable Algorithms**

_Manhattan Distance_: Used to find the services closest to a student.

_Sorting & Ranking_: Star rating system with rounded averages and dynamic sorting by relevance.
